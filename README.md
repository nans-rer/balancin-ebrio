# balancin-ebrio
Robot balancin correspondiente al proyecto final del curso de Mecatronica ME4250-2, semestre de primavera, de la Universidad de Chile. 

Integrantes:
- Camilo Urra
- Hernán Reyes
- Joaquín Gutiérrez
- Luisfelipe Muñoz
- Marcio Ramírez

El diseño es de una persona ebria apoyandose en un poste para mantener su equilibrio.
